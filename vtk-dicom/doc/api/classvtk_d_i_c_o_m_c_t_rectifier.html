Detailed Description

Prepare a CT for 3D processing.

This class will identify gantry-tilted CT images and resample them into a rectangular volume. This is often a necessary step prior to volume rendering or other forms of 3D rendering.

static double vtkDICOMCTRectifier::GetGantryDetectorTilt ( vtkMatrix4x4 *  volumeMatrix)
static

A static method to measure the tilt from a matrix.

This can be used to see whether it is necessary to rectify the volume. It returns the Gantry Detector Tilt angle, in degrees, as computed from the shear in the volume matrix.
