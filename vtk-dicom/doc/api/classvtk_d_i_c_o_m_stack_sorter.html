Detailed Description

Sort a DICOM series into stacks.

Given a vtkDICOMMetaData object for a DICOM series, this class will index the files and frames in the series according to what stack they belong to. A stack is a contiguous set of slices that form a volume. Most DICOM series consist of only a single stack, but sometimes multiple stacks are acquired within one series.
