Detailed Description

Generate DICOM data objects for MR images.

Generate a DICOM data set belonging to one of the MR SOP classes. The assumption is that images have been read into VTK as MR, and are being written out as derived images after being processed. The specific IOD classes supported are as follows:

  • MR Image, 1.2.840.10008.5.1.4.1.1.4
