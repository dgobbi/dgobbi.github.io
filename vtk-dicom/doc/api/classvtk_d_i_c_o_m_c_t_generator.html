Detailed Description

Generate DICOM data objects for CT images.

Generate a DICOM data set belonging to one of the CT SOP classes. The assumption is that images have been read into VTK as CT, and are being written out as derived images after being processed. The specific IOD classes supported are as follows:

  • CT Image, 1.2.840.10008.5.1.4.1.1.2
